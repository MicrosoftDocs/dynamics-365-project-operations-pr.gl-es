<span data-ttu-id="3bae4-106">Os compoñentes incluídos están suxeitos a:</span><span class="sxs-lookup"><span data-stu-id="3bae4-106">Included components are subject to:</span></span>

  - <span data-ttu-id="3bae4-113">Non imputable</span><span class="sxs-lookup"><span data-stu-id="3bae4-113">Non-chargeable</span></span>

<span data-ttu-id="3bae4-114">Os compoñentes imputables poden definirse en tarefas, roles e categorías de transacción.</span><span class="sxs-lookup"><span data-stu-id="3bae4-114">Chargeable components can be defined on tasks, roles, and transaction categories.</span></span>

### <a name="update-a-project-task-to-be-chargeable-or-non-chargeable"></a><span data-ttu-id="3bae4-121">Actualizar unha tarefa de proxecto para que sexa imputable ou non imputable</span><span class="sxs-lookup"><span data-stu-id="3bae4-121">Update a project task to be chargeable or non-chargeable</span></span>
