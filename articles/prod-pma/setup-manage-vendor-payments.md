# <a name="set-up-and-use-pay-when-paid-vendor-payments"></a><span data-ttu-id="12aef-103">Configurar e usar pagamentos de fornecedores que se pagan ao recibir o pagamento</span><span class="sxs-lookup"><span data-stu-id="12aef-103">Set up and use pay-when-paid vendor payments</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="12aef-104">Cando vostede aprobe que un vendedor traballe como subcontratista, quizais queira reter o pagamento ata que o seu cliente lle pague o proxecto.</span><span class="sxs-lookup"><span data-stu-id="12aef-104">When you approve a vendor to work as a subcontractor, you might want to withhold payment to the vendor until your customer pays you for the project.</span></span> <span data-ttu-id="12aef-105">Para permitir esta situación, pode configurar termos de pagamento ao recibir o pagamento (PWP) cando configura o pedido de compra (PO) co fornecedor.</span><span class="sxs-lookup"><span data-stu-id="12aef-105">To support this scenario, you can set up pay-when-paid (PWP) terms when you set up the purchase order (PO) with the vendor.</span></span>

<span data-ttu-id="12aef-106">Por exemplo, cando un cliente paga un importe dunha factura do proxecto, pode liberar parte ou a totalidade do importe da factura do fornecedor.</span><span class="sxs-lookup"><span data-stu-id="12aef-106">For example, when a customer pays an amount on a project invoice, you can release some or all of the vendor invoice amount.</span></span> <span data-ttu-id="12aef-107">Simplemente configure os termos de PWP que especifiquen que se pagará ao fornecedor despois de recibir unha porcentaxe do pagamento relacionado do cliente.</span><span class="sxs-lookup"><span data-stu-id="12aef-107">Just set up PWP terms that specify that the vendor will be paid after you receive a percentage of the related payment from the customer.</span></span> <span data-ttu-id="12aef-108">Se recibe un pagamento parcial dun cliente, pode liberar manualmente algunhas das facturas do fornecedor relacionadas.</span><span class="sxs-lookup"><span data-stu-id="12aef-108">If you receive partial payment from a customer, you can manually release some of the related vendor invoices for payment.</span></span>

<span data-ttu-id="12aef-109">O seguinte exemplo describe o proceso cando se usan termos de PWP.</span><span class="sxs-lookup"><span data-stu-id="12aef-109">The following example outlines the process when PWP terms are used.</span></span>

## <a name="example"></a><span data-ttu-id="12aef-110">Exemplo</span><span class="sxs-lookup"><span data-stu-id="12aef-110">Example</span></span>

<span data-ttu-id="12aef-111">A súa organización acepta subministrar a un cliente 100 ordenadores que teñen instalado software por un prezo de 150,00 dólares estadounidenses (USD) por ordenador.</span><span class="sxs-lookup"><span data-stu-id="12aef-111">Your organization agrees to provide a customer with 100 computers that have software installed, for a price of 150.00 US dollars (USD) per computer.</span></span> <span data-ttu-id="12aef-112">Logo contrata a un fornecedor para que lle subministre os ordenadores que teñen instalado o software.</span><span class="sxs-lookup"><span data-stu-id="12aef-112">You then hire a vendor to provide you with the computers that have software installed.</span></span> <span data-ttu-id="12aef-113">Segundo o contrato, o cliente debe aprobar a calidade dos ordenadores antes de pagarlle á súa organización.</span><span class="sxs-lookup"><span data-stu-id="12aef-113">According to the agreement, the customer must approve the quality of the computers before it pays your organization.</span></span> <span data-ttu-id="12aef-114">A política da súa organización é reter o pagamento aos fornecedores ata que o cliente lle pague.</span><span class="sxs-lookup"><span data-stu-id="12aef-114">Your organization's policy is to withhold payment to vendors until the customer has paid you.</span></span> <span data-ttu-id="12aef-115">Polo tanto, configurou o proxecto para que teña unha porcentaxe de PWP do 100 por cento.</span><span class="sxs-lookup"><span data-stu-id="12aef-115">Therefore, you set up the project so that it has a PWP percentage of 100 percent.</span></span>

<span data-ttu-id="12aef-116">O fornecedor envíalle os 100 ordenadores que teñen instalado o software, xunto cunha factura por 10 000,00 USD.</span><span class="sxs-lookup"><span data-stu-id="12aef-116">The vendor sends you the 100 computers that have software installed, together with an invoice for 10,000.00 USD.</span></span> <span data-ttu-id="12aef-117">Debido a que os termos de PWP están configurados para o seu proxecto, non paga ao fornecedor unha vez recibidos os ordenadores.</span><span class="sxs-lookup"><span data-stu-id="12aef-117">Because PWP terms are set up for your project, you don't pay the vendor upon receipt of the computers.</span></span> <span data-ttu-id="12aef-118">En vez diso, envía os ordenadores ao cliente, xunto cunha factura por 15 000.00.</span><span class="sxs-lookup"><span data-stu-id="12aef-118">Instead, you send the computers to the customer, together with an invoice for 15,000.00.</span></span> <span data-ttu-id="12aef-119">O cliente inspecciona os ordenadores e aproba o importe total da factura do proxecto.</span><span class="sxs-lookup"><span data-stu-id="12aef-119">The customer inspects the computers and approves the full amount of the project invoice.</span></span>

<span data-ttu-id="12aef-122">Cando configura os termos de PWP para un proxecto, debe especificar, como porcentaxe, a cantidade mínima que un cliente lle debe pagar polo proxecto antes de pagarlle ao fornecedor.</span><span class="sxs-lookup"><span data-stu-id="12aef-122">When you set up PWP terms for a project, you must specify, as a percentage, the minimum amount that a customer must pay you for the project before you will pay the vendor.</span></span> <span data-ttu-id="12aef-123">O importe limiar calcúlase automaticamente nas facturas do cliente para o proxecto.</span><span class="sxs-lookup"><span data-stu-id="12aef-123">The threshold amount is automatically calculated on the customer invoices for the project.</span></span> <span data-ttu-id="12aef-124">Siga estes pasos para configurar a porcentaxe limiar para os termos de PWP.</span><span class="sxs-lookup"><span data-stu-id="12aef-124">Follow these steps to set up the threshold percentage for PWP terms.</span></span>
